NailService - это CRM система для небольших салонов красоты/парикмахерских/ногтевых студий.
Позвояет вести запись клиентов на прием, учитывать статистику стоимости оказанных услуг, периодичности посещений и т.д.

В настоящее время находится в процессе разработки.

Что уже реализовано:

- Сам сервис (серверная часть) и БД (MS SQL) разнесены на разные хосты. 
- БД поднята в докер-контейнере.
- Реализовано логгирование HTTP запросов к серверу (используется NLog).
- Реализован REST API.
- Реализованы CRUD операции при работе с БД.
